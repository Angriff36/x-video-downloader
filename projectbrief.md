# Project Brief

## Overview
Building a Flutter + FastAPI video downloader app targeting mobile (Android), with a planned desktop version. The app allows users to download videos from X (Twitter), Rumble, and other platforms with native media storage, Play Store compatibility, and simple UI.

## Core Features
- Video download from X and Rumble
- Local-only FastAPI backend using yt-dlp + ffmpeg
- Device storage integration
- Share intent support
- Android + desktop platform support (planned)

## Target Users
Mobile users who want to download and manage videos from Twitter/X and Rumble without complex tools or web scraping.

## Technical Preferences
- Backend: FastAPI, Python, yt-dlp, ffmpeg
- Frontend: Flutter
- Deployment: Android APK, future desktop app
- Local-first architecture for compliance and privacy
