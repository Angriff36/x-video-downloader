# Product Context

## Problem
Users struggle to download and store videos from social media platforms like X (Twitter) and Rumble, especially on mobile devices. Many tools are web-based or unreliable, and browser extensions often break.

## Solution
A mobile-only app with a clean interface that uses a local backend (FastAPI + yt-dlp) to download and save videos directly to the device. Ensures performance, reliability, and compatibility with Play Store policies.

## User Experience Goals
- Paste link → press download → view result
- Background download with status tracking
- Gallery integration to view saved videos
